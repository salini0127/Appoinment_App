# Appoinment_App

# 🗓️ CapMinds Appointment Scheduler

## 📌 Project Overview
This is a responsive Appointment Scheduling Web Application developed as part of the CapMinds Technical Task.

The application allows users to:
- View appointments in a calendar layout
- Book new appointments
- Edit and delete appointments
- View all appointments in a dashboard

---

## 🚀 Features

✅ Calendar View (Monthly layout)  
✅ Book Appointment (Modal popup)  
✅ Edit & Delete Appointments  
✅ Dashboard with Appointment Table  
✅ LocalStorage Data Persistence  
✅ Responsive Design  

---
